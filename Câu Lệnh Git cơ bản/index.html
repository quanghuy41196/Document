Câu Lệnh Cơ Bản



git config

 Dùng để set user name và email của bạn trong main configuration file.
Cách dùng : Để kiểm tra tên và kiểu email trong cấu hình dùng git config -- global user.name và git config -- global user.email. Để set email hoặc tên mới git config -- global user.name = “Quang Huy” và git config -- global user.email = “quanghuy41196@gmail.com”

git init

 Dùng để khởi tạo 1 git repository 1 project mới hoặc đã có.
Cách dùng : git init trong thư mục gốc của dự án.

git clone

 Dùng để Copy 1 git repository từ remote source.
Cách dùng : git clone <:clone git link url:>

git status

 Dùng để check trạng thái của những file bạn đã thay đổi trong thư mục làm việc. VD: Tất cả các thay đổi cuối cùng từ lần commit cuối cùng.
Cách dùng : git status trong thư mục làm việc.

git add

 Dùng để hêm thay đổi đến stage/index trong thư mục làm việc.
Cách dùng : git add .

git commit

 Dùng để ghi lại việc thêm/ thay đổi file hay thư mục vào repository
Cách dùng : git commit -m ”comment đã thêm/ thay đổi để quản lý”


Câu Lệnh Chưa Sử Dụng



git push/git pull

 Dùng để Push hoặc Pull các thay đổi đến remote. Nếu bạn đã added và committed các thay đổi và bạn muốn đẩy nó lên hoặc remote của bạn đã update và bạn apply tất cả thay đổi đó trên code của mình.
Cách dùng : git pull <:remote:> <:branch:> and git push <:remote:> <:branch:>

git branch

 Dùng để liệt kê tất cả các branch (nhánh).
Cách dùng : git branch hoặc git branch -a

git checkout

 Dùng để Chuyển sang branch khác
Cách dùng : git checkout <: branch:> hoặc ** _ git checkout -b <: branch:> nếu bạn muốn tạo và chuyển sang một chi nhánh mới

git stash

 Dùng để lưu thay đổi mà bạn không muốn commit ngay lập tức.
Cách dùng : git stash trong thư mục làm việc của bạn.

git merge

 Dùng để hợp nhất 2 branch lại với nahu.
Cách dùng : Chuyển tới branch bạn muốn merge rồi dùng git merge <:branch_ban_muon_merge:>

git reset

 Tác dụng: Bạn đã đưa một tập tin nào đó vào Staging Area nhưng bây giờ bạn muốn loại bỏ nó ra khỏi đây để không phải bị commit theo.
Cách dùng : git reset HEAD tên_file

git remote

 Tác dụng: Để check remote/source bạn có hoặc add thêm remote
Cách dùng: git remote để kiểm tra và liệt kê. Và git remote add <: remote_url:> để thêm.

git add

 Dùng để đưa một tập tin vào Staging Area
Cách dùng: git add tên_file hoặc muốn thêm hết file của thư mục thì git add all
