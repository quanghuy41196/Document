Kiểu dữ liệu



Kiểu number

 Là các số bất kỳ loại nào: số nguyên hoặc dấu phẩy động.

Exam

var x = 20; // gán giá trị cho x = 20, vậy x là kiểu Number
var y = 20,234; // gán giá trị cho y = 20,234 vậy y là kiểu Number
console.log(x); //Hiện thị x ra màn hình console của trình duyệt
console.log(y); //Hiện thị y ra màn hình console của trình duyệt
console.log(x+y); //Hiện thị tổng x + y ra màn hình console của trình duyệt

Kiểu string

 Chuỗi có thể có một hoặc nhiều ký tự, không có loại ký tự đơn riêng biệt.

Exam

var x = "con ga" // x được gán giá trị là "con ga", vậy x là kiểu String
var y = 'mai' // y được gán giá trị là "mai", vậy x là kiểu String

Kiểu boolean

 boolean nó chỉ trả về 2 giá trị là ” True ” or ” False ”

Exam

var x = true; // x được gán giá trị cho true, vậy x là kiểu Boolean
var y = false; // y được gán giá trị cho false, vậy y là kiểu Boolean
var m = 1;
var n = 2;
console.log(m = n); // Hiện thị ra màn hình console của trình duyệt kết quả m = n, ở đây nó sẽ trả về false

Kiểu Null

 Nó chỉ là một giá trị đặc biệt có ý nghĩa “không có gì”, “trống rỗng” hoặc “giá trị không rõ”.

Exam

var n = null; // n được gán giá trị cho null, vậy n là kiểu Null
console.log(n); // Hiện thị ra màn hình console của trình duyệt giá trị của n, ở đây nó sẽ trả về null

Kiểu Underfined

 Là biến chưa được có giá trị.

Exam

var x; // x chưa được gán giá trị nào
console.log(x); //Hiện thị ra màn hình console giá trị của x, ở đây kết quả trả về là underfined

Kiểu Object

 Là kiểu dữ liệu đối tượng, một đối tượng ở đây là chỉ một đối tượng thật trong cuộc sống. Ví dụ một chiếc xe tăng, một con mèo, một lá cờ Đảng … Đối tượng được chia làm 2 phần là: thuộc tính và phương thức.

Exam

var meo = { ten: 'meomeo', maulong: 'den', tuoi: 3 };
// Khởi tạo một object là meo có tên là 'meomeo' màu lông là 'den' và có tuổi là 3
console.log(meo.ten); // Hiện thị ra màn hinh console của trình duyệt ten của meo, ở đây nó sẽ trả về 'meomeo'
